    <b>rocketCast is a Progressive Web App that brings the best tech podcasts closer to you, with a built-in player and episode details.</b>
</p>

<p align="center">
    The application was built during Next Level Week #05 organized by Rocketseat, showcasing modern web technologies to build a full-featured PWA podcast platform.

<h2 id="features">✨ Features</h2>

- Browse a list of Faladev podcast episodes
- View episode details (title, description, duration)
- Built-in audio player with playback controls
- Shuffle, repeat, previous and next episode controls
- Progressive Web App (installable on any device)
- Smooth UI animations with Framer Motion
- Fully responsive design
- Dark theme

<h2 id="technologies">💻 Technologies</h2>

<h2 id="started">🚀 Getting started</h2>

<h3>Cloning</h3>

```bash
git clone https://github.com/MarlonVictor/rocketCast.git
cd rocketCast
```

<h3>Installation</h3>

```bash
yarn install
```

<h3>Starting</h3>

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

<h3>Deployment</h3>

The app is deployed at [https://rocketcast.vercel.app/](https://rocketcast.vercel.app/).

To build for production:

```bash
yarn build
```

The output is generated in the `.next/` folder and can be deployed to any platform that supports Next.js.

<h2 id="usage">👀 Usage</h2>

1. Browse the episode list on the home page.
2. Click on an episode to load it in the player.
3. Use the player controls to play, pause, skip, shuffle or repeat.
4. Install the app as a PWA for an offline-friendly experience.
